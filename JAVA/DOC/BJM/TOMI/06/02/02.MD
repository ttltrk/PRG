Java Solution

WHILE

```java
public class HelloWorld{

     public static void main(String []args){
         
        int index = 0;
        int result = 0;
          
        while(index <= 4)
          {
          System.out.println("Hello World");
          index++;
          }
        
     }
}

>>>
Hello World

Hello World
>>>
```

---

DO_WHILE

```java
public class HelloWorld{

     public static void main(String []args){
         
        int index = 0;
        int result = 0;
          
        do
          {
          System.out.println("Hello World");
          index++;
          } while(index <= 4);
        
     }
}

>>>
Hello World
Hello World
Hello World
Hello World
Hello World
>>>
```
