### 01.05.02 - Addition and Subtraction

---

Addition

```java
The + operator adds together two values, such as two constants, 
a constant and a variable, or a variable and a variable.

int sum1 = 50 + 10; 
int sum2 = sum1 + 66; 
int sum3 = sum2 + sum2;
```

---

Subtraction

```java
The - operator subtracts one value from another.

int sum1 = 1000 - 10;
int sum2 = sum1 - 5;
int sum3 = sum1 - sum2;
```
