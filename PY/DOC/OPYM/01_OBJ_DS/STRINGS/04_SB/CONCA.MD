### Concatenation

---

```python
str1 = 'Hello'
str2 ='World!'

print('str1 + str2 = ', str1 + str2)

print('str1 * 3 =', str1 * 3)

>>>
str1 + str2 =  HelloWorld!
str1 * 3 = HelloHelloHello
>>>
```

```python
s = 'Hello World concatenate me!'

print(s.upper())
print(s.lower())
print(s.split())
print(s.split('W'))

>>>
HELLO WORLD CONCATENATE ME!
hello world concatenate me!
['Hello', 'World', 'concatenate', 'me!']
['Hello ', 'orld concatenate me!']
>>>
```
