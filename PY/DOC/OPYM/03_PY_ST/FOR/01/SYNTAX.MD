### Syntax for for loops

---

```python
for val in sequence:
	Body of for
```

---

```python
numbers = [6, 5, 3, 8, 4, 2, 5, 4, 11]

sum = 0

for val in numbers:
	sum = sum+val

print("The sum is", sum)

>>>
```
