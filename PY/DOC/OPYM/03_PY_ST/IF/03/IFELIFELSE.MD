### if...elif...else Statement

---

```python
if test expression:
    Body of if
elif test expression:
    Body of elif
else: 
    Body of else
```

---

```python
num = 3.4

if num > 0:
    print("Positive number")
elif num == 0:
    print("Zero")
else:
    print("Negative number")
    
>>>
Positive number
>>>
```
