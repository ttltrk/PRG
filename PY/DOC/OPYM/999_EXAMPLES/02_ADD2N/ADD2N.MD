
Add two numbers
======

[<<<]()

In this program, you will learn to add two numbers and display it using print() function.

```python
# This program adds two numbers

num1 = 3.4
num2 = 4.3

# Add two numbers
sum = float(num1) + float(num2)

# Display the sum
print('The sum of {0} and {1} is {2}'.format(num1, num2, sum))

>>>

>>>
```
