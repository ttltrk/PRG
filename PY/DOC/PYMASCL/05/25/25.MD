### 25 - Storing items in Variables

---

```python
greeting = "hello"
name = "trk"

print(greeting + ' ' + name)

>>>
hello trk
>>>
```

---

```python
a = 12
b = 3

print(a+b)
print(a-b)
print(a*b)
print(a/b)
print(a//b)
print(a%b)

>>>
15
9
36
4.0
4
0
>>>
```
